<p>Three years ago, <a href="https://github.com/rstudio/tinytex-releases/issues/37">Cole Arendt requested</a> pre-built TinyTeX binaries for ARM Linux. Today, I&rsquo;m happy to announce that we now provide pre-built TinyTeX binaries for <code>arm64</code> and <code>musl</code>-based Linux (e.g. Alpine) systems. On these platforms you no longer need to build TinyTeX from source; binary installation will be much faster.</p>
<p>We also updated the release filename convention to include OS and architecture, so release assets are clearer and easier to work with.</p>
